# Introduction
 
 Proyecto de extension chrome para gestion de favoritos echo con vuejs 

# How to start

```bash
git clone git@github.com:oillescas/chorme-my-bookmarks.git
cd chorme-my-bookmarks
npm i   # or `npm run reinstall` if you get an error
npm start    # start app
```
